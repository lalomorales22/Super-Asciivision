Open Source Desktop AI Shell

Super ASCIIVision

Chat, generate images and video, edit media, write code, manage terminals, play music, and control it all from your phone — powered by xAI and Ollama, in one desktop app.

Install GitHub API Docs

Everything in one shell

Chat

Streamed conversations with xAI or Ollama. Workspace-backed context, agentic tool use — file read/write, shell commands, search.

Image & Video

Generate images and video with xAI Grok. Category-organized gallery with drag-and-drop import.

Voice & Audio

Text-to-speech generation and live realtime voice chat. Push-to-talk or hands-free mode.

Media Editor

Timeline-based editor with visual, audio, subtitle, and overlay tracks. Export via ffmpeg.

IDE

File explorer, code editor with syntax highlighting, AI copilot, Quick Open, and browser preview.

Tiles

1x2, 2x2, or 3x3 grid of independent PTY terminal sessions. Persistent across navigation.

Music

Built-in player for your local library. MP3, WAV, OGG, FLAC, M4A. Playlists, shuffle, repeat, mini-player.

Hands

Pair your phone over WiFi with zero setup. Chat, generate media, and access your terminal remotely.

ASCIIVision Terminal

A full terminal experience built in Rust with ratatui. Multi-provider AI chat, ASCII video playback, live webcam, 3D effects, system monitoring, games, and tiling — all inside the app.

Install

One command. The script handles all dependencies, builds both apps, and installs.

macOS

git clone https://github.com/lalomorales22/Super-Asciivision.git
cd Super-Asciivision
./install.sh

Requires macOS 13+. Installs Homebrew, Node.js, Rust, FFmpeg, LLVM if missing. Builds and copies to /Applications.

Linux

git clone https://github.com/lalomorales22/Super-Asciivision.git
cd Super-Asciivision
./install-linux.sh

Tested on Ubuntu 22.04/24.04, Fedora, Arch. Installs system deps, Node.js, Rust, Ollama. Builds and installs to ~/.local/bin.

Development

git clone https://github.com/lalomorales22/Super-Asciivision.git
cd Super-Asciivision
npm install
./build-asciivision.sh
npm run tauri dev

Hot-reloading dev server on localhost:1420. Requires Node.js 20+, Rust, FFmpeg dev libs, LLVM.

API Documentation

The docs hub is a self-contained PHP app with the full API reference, development history, and an AI chat assistant powered by Ollama that can answer questions about the codebase.

Run the docs server

Start a PHP dev server from the docs directory and open it in your browser:

cd docs
php -S localhost:8000

Then visit localhost:8000. The AI chat feature requires Ollama running locally.
